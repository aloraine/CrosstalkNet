Introducing CrosstalkNet
App Documentation

App Description

CrosstalkNet is a web app focused on exploring the interactions between tumor epithelial and stromal cells in a bipartite manner. It has multiple utilities that allow for exploring different levels of neighbours, determining if there are paths between two genes of interest, and finding out what genes are highly connected.

Requirements

  • Web Browser: Chrome officially supported, others work too
  • An account to upload files
  • Nothing else!

Choosing Data

In order to begin exploring interactions, a network type must be selected via the radio buttons. Once a network type is selected, the corresponding file(s) need to be selected as well.

Uploading Data

If a user wants to use their own data, they are able to upload it in the Data sub-tab of the Main Graph tab. A user needs to be logged in for this feature to be enabled.

Requirements

  • Files must contain a single dgCMatrix from the Sparse Matrix package
  • No matrix is allowed to have NA values
  • The rownames of a matrix must match its colnames
  • The rownames and colnames of a matrix must not contain NA values
  • Files must be saved using the saveRDS function
  • Files must have an Rdata extension
  • Files must be smaller than 20MB in size
A dialog box will open shortly indicating the success status of the upload.

Main Graph

Here is a video guide showing the basic usage of the Main Graph tab.

Features

  • Investigate multiple genes at once
  • View first neighbours
  • View second neighbours
  • Filter interactions
  • Locate genes in graph
  • Download interactions

Usage

Intended to view clusters of first and second neighbours for multiple genes of interest. Can show if genes have interactors in common and provides a focused subnetwork of the overall network that has possibly millions of interactions.

Interaction Explorer

Here is a video guide showing the basic usage of the Interaction Explorer tab.

Features

  • Focus on one gene of interest
  • View infinite levels of neighbours
  • Locate genes in graph
  • Download interactions

Usage

Intended to explore potential pathways from a single gene of interest through its neighbours. At each level of neighbours, a different gene is selected and its corresponding neighbours will be added to the chain of neighbours.

Path Explorer

Here is a video guide showing the basic usage of the Path Explorer tab.

Features

  • Find all paths between 2 genes
  • Sort by various columns
  • Download list of paths

Usage

Retrieves the path between an epi and stroma gene if one exists. Retrieves all paths between two stroma genes, or two epi genes.

Degree Explorer

Here is a video guide showing the basic usage of the Degree Explorer tab.

Features

  • View top genes ordered by degree
  • View genes having specified degree or higher
  • Download list of paths

Usage

Retrieves all genes are their corresponding degrees in an ordered manner with cutoffs available for either minimum degree or maximum amount of genes.

Community Explorer

Here is a video guide showing the basic usage of the Community Explorer tab.

Features

  • Visualize all communities at once
  • Examine members of specific commnities
  • Download communities

Usage

Visualizes all genes clustered by their communities. Individual communities can be looked at in tabular form to inspect their members.
