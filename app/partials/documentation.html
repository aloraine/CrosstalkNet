Introducing CrosstalkNet
App Documentation

App Description

Crosstalk Net is a data mining app focused on examining the interactions between tumor epithelial and stromal cells. It has multiple utilities that allow for exploring different levels of neighbours, determining if there are paths between two genes of interest, and finding out what genes are highly connected.

Requirements

  • Google Chrome
  • An account to upload files
  • Nothing else!

Choosing Data

In order to begin exploring interactions, a network type must be selected via the radio buttons. Once a network type is selected, the corresponding file(s) need to be selected as well.

Uploading Data

If a user wants to use their own data, they are able to upload it in the Data sub-tab of the Main Graph tab. A user needs to be logged in for this feature to be enabled.

Requirements

  • Files must contain a single dgCMatrix from the Sparse Matrix package
  • No matrix is allowed to have NA values
  • The rownames of a matrix must match its colnames
  • The rownames and colnames of a matrix must not contain NA values
  • Files must be saved using the saveRDS function
  • Files must have an Rdata extension
  • Files must be smaller than 20MB in size
A dialog box will open shortly indicating the success status of the upload
