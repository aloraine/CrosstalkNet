Introducing CrosstalkNet

App Description

Crosstalk Net is a data mining app focused on examining the interactions between tumor epithelial and stromal cells. It has multiple utilities that allow for exploring different levels of neighbours, determining if there are paths between two genes of interest, and finding out what genes are highly connected.

 Crosstalk Net is a data mining app focused on examining the interactions between tumor epithelial and stromal cells. Getting Started
 When the app first loads, users will be in the Data sub-tab of the Main Graph tab. This sub-tab allows users to choose the type of network they want to examine: Normal, Tumor, or Delta. It also allows users to upload their own data files for a particular network type. Once a user has selected the appropriate file(s), clicking on the Get Genes button will enable the other tabs.
